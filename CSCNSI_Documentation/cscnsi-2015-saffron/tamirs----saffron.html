TAMIRS - Lustre

Installing Lustre: 

Download SRPM: LINK

# rpmbuild --rebuild lustre-2.6.0-2.6.32_431.20.3.el6_lustre.x86_64.src.rpm

# mv *.rpm  /local-updates-saffron/RPMS/x86_64/

# cd /local-updates-saffron/RPMS/x86_64/

# createrepo .

Be sure to download e2fsprogs versions: LINK AND all the dependancies!!!!

Run createrepo everytime you add new rpms to /local-updates-saffron/RPMS/x86_64/ 

Update /etc/yum.conf (master/server side):

# Saffron localupdates
[Saffron]
name=Saffron
baseurl=file://saffron-master.ccstar.lanl.gov/local-updates-saffron/RPMS/x86_64/
failovermethod=priority
enabled=1

Update /etc/yum.conf (client side):

# Saffron localupdates
[Saffron]
name=saffronLocal
baseurl=file:///local-updates-saffron/RPMS/x86_64
failovermethod=priority
enabled=1

Rebuild Images (do for oss, mgs, and client images -- watch for errors on build!!!):

/opt/ppsst/bin/build -f -v -c /var/lib/perceus/vnfs/oss -v
/var/lib/perceus/vnfs/oss/createimage

REBOOT NODES TO GET NEW IMAGE!!!!





Configuring Lustre (using HDD):

MGS [sa01]

# mkfs.lustre --fsname=safflust --mgs --mdt --index=0 /dev/sda1

# mkdir /mnt/mds

# mount -t lustre /dev/sda1 /mnt/mds


NVMe device --> /dev/nvme0n1p1
Kove device RAID 0 --> /dev/md0



OSS [sa02-sa05]


# mkfs.lustre --reformat --ost --fsname=safflust --index 0 --mgsnode=10.142.0.1@o2ib /dev/sda1

Use this with index X with x as an increasing integer!!!! 

# mkfs.lustre --ost --fsname=safflust --index X --mgsnode=10.142.0.1@o2ib /dev/sda1

# mkdir /mnt/mds

# mount -t lustre /dev/sda1 /mnt/mds


CLIENT [sa06-sa10]

# mkdir /mnt/lustre

# mount -t lustre 10.142.0.1@o2ib:/safflust /mnt/lustre
