debugfs:

LINK

CONFIGURE

Need to mount: 
# mount -t debugfs nodev /sys/kernel/debug

This gives access to the directory: 

# cd /sys/kernel/debug/tracing/

In this directory available_filter_functions can be grep'd for desired parameters for tracing

# cat available_filter_functions | grep getattr

Also have interest in: create open link unlink readdir getattr setattr close

The desired list should append to the set_ftrace_filter

# cat available_filter_functions | grep getattr >> set_ftrace_filter


RUN:

Check that the set-ftrace_filter file has what is desired

# cat set_ftrace_filter

Ensure tracing is enabled and start the trace:

# echo 1 > traceing_enabled

# echo 1 > tracing_on

To stop the trace

# echo 0 > tracing_on

To see the trace look at /sys/kernel/debug/tracing/trace
