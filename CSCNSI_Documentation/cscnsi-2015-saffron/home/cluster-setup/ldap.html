Create base tree with group
/root/ldap/base.ldif
dn: dc=saffron,dc=com
objectClass: dcObject
objectClass: organization
o: saffron.com
dc: saffron

dn: ou=Trainers,dc=saffron,dc=com
objectClass: organizationalUnit
objectClass: top
ou: Trainers

dn: ou=GymLeaders,dc=saffron,dc=com
objectClass: organizationalUnit
objectClass: top
ou: GymLeaders

Add to database
# ldapadd -D cn=Manager,dc=saffron,dc=com -w password -f base.ldif

Create a User
/root/ldap/sabrina.ldif
dn: uid=sabrina,ou=Trainers,dc=saffron,dc=com
objectClass: top
objectClass: account
objectClass: posixAccount
objectClass: shadowAccount

Add the user to a group
/root/ldap/sabrinagroup.ldif
dn: cn=sabrina,ou=GymLeaders,dc=saffron,dc=com
objectClass: top
objectClass: posixGroup

Add to database
# ldapadd -D cn=Manager,dc=saffron,dc=com -w password -f sabrinagroup.ldif

Services
Host
# chkconfig slapd on
# chkconfig nslcd on
# service nslapd start
# service nslcd start

Internal
# chkconfig nslcd on
# service nslcd start
 
Configure ldap (both)
# authconfig --enableldap --enableldapauth --ldapserver=ldap.saffron.com --ldapbasedn="dc=saffron,dc=com" --enablemkhomedir --update
