Configuration

Server

Install the following packages (just to be safe. I don't actually remember doing this, but hey, why not?):

yum install openmpi blas atlas

Download the latest version of HPL:

wget http://www.netlib.org/benchmark/hpl/hpl-2.1.tar.gz

Untar the file. Note: we downloaded the file into the /home directory. It doesn't matter where you put it, but be sure to change the commands to reflect your paths.

tar -xvf  /home/hpl-2.1.tar.gz

Copy the Make.Linux_PII_CBLAS file from /home/hpl-2.1/setup/ to /home/hpl-2.1/
cp /home/hpl-2.1/setup/Make.Linux_PII_CBLAS /home/hpl-2.1/Make.Linux_PII_CBLAS 
Edit the file. The following is our entire file (minus all comments). You might have to change it; our paths might not the be same as yours.

SHELL        = /bin/sh
#
CD           = cd
CP           = cp
LN_S         = ln -s
MKDIR        = mkdir
RM           = /bin/rm -f
TOUCH        = touch

...

ARCH         = Linux_PII_CBLAS

...

TOPdir       = /home/hpl-2.1
INCdir       = $(TOPdir)/include
BINdir       = $(TOPdir)/bin/$(ARCH)
LIBdir       = $(TOPdir)/lib/$(ARCH)
#
HPLlib       = $(LIBdir)/libhpl.a

...

MPdir        = /usr
MPinc        = -I$(MPdir)/include/openmpi-x86_64
MPlib        = $(MPdir)/lib64/openmpi/lib/libmpi.so

...

LAdir        = /usr/lib64/atlas
LAinc        =
LAlib        = $(LAdir)/libcblas.so.3 $(LAdir)/libatlas.so.3

...

F2CDEFS      =

...

HPL_INCLUDES = -I$(INCdir) -I$(INCdir)/$(ARCH) $(LAinc) $(MPinc)
HPL_LIBS     = $(HPLlib) $(LAlib) $(MPlib)

...

HPL_OPTS     = -DHPL_CALL_CBLAS
HPL_DEFS     = $(F2CDEFS) $(HPL_OPTS) $(HPL_INCLUDES)

...

CC           = /usr/lib64/openmpi/bin/mpicc
CCNOOPT      = $(HPL_DEFS)
CCFLAGS      = $(HPL_DEFS) -fomit-frame-pointer -O3 -funroll-loops

...

LINKER       = /usr/lib64/openmpi/bin/mpicc
LINKFLAGS    = $(CCFLAGS)
#
ARCHIVER     = ar
ARFLAGS      = r
RANLIB       = echo


The find / | grep <query> command will help you a lot in finding where everything is located and figuring out the correct paths.

After modifying the file, compile it:

make arch=Linux_PII_CBLAS

If you get no errors then you've done it! Celebrate. Now you can run LinPack:

cd ~/hpl-2.1/bin/Linux_PII_CBLAS
mpirun -np 200 ./xhpl

You can edit the configuration for LinPack with the HPL.dat file found in /home/hpl-2.1/bin/Linux_PII_CBLAS/HPL.dat

vim /home/hpl-2.1/bin/Linux_PII_CBLAS/HPL.dat
