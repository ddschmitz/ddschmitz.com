fail2ban

Descriptions: fail2ban is a service to reduce the chances of brute force login attempts.

Installation:
# yum install fail2ban
# cp /etc/fail2ban/jail.conf /etc/fail2ban/jail.local

Configuration:
For configuration options look at:
# emacs /etc/fail2ban/jail.local

# service fail2ban start
